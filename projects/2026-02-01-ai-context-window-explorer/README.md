# AI Context Window Explorer (Feb 1, 2026)

## About

An educational project that explains what context windows are in AI language models, why they matter, and how different models compare in their context capacity.

## Features

- **Educational Explanation**: AI-powered explanation of what context windows are and their importance
- **Model Comparison**: Compare context window sizes across popular AI models (OpenAI, Anthropic, Google, Meta, etc.)
- **Token Calculator**: Analyze your text to see how much of a model's context window you're using
- **Visual Explanation**: AI-generated visualization of how context windows work

## Technology

- Frontend: Vanilla HTML/CSS/JavaScript
- Backend: Python (Vercel serverless function)
- AI: OpenRouter (free models) and Pollinations.AI for image generation
- Architecture: Single API router with date-based project routing

## API Actions

- `start` - Get basic explanation of context windows
- `models` - Get comparison table of models' context sizes
- `calculate` - Analyze text and model to compute token usage percentage
- `visualize` - Generate an image visualizing the concept
