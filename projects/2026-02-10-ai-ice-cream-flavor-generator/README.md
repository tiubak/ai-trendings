# AI Ice Cream Flavor Generator

Generate creative and delicious ice cream flavor ideas with AI. Enter optional hints (like chocolate, fruit, or a theme), dietary preferences, and season to get a unique flavor profile complete with:

- A creative flavor name
- Tasty description of texture and experience
- List of key ingredients
- A fun fact about the flavor
- Appetizing image generated with Pollinations.AI

This project demonstrates using OpenRouter for structured JSON generation and Pollinations.AI for image generation.

## Technical Details

- **Category:** Fun
- **Date:** 2026-02-10
- **APIs:** OpenRouter (text), Pollinations.AI (image)
- **Frontend:** Pure HTML/CSS/JavaScript with fetch API
