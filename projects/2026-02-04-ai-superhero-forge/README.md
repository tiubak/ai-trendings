# AI Superhero Forge (2026-02-04)

Create your own superhero with AI! Generate a unique character with a name, powers, backstory, weakness, and visual appearance.

## Features

- **Generate Superhero**: Provide an optional name and theme to customize your hero, or leave blank for a completely random character.
- **AI-Powered**: Uses OpenRouter for text generation and Pollinations.AI for image generation.
- **Interactive UI**: Simple form with instant results.

## Usage

1. Open `index.html` in a browser (served through the AI Trendings app).
2. Optionally enter a superhero name and select a theme.
3. Click "Generate Superhero" or "Generate Random".
4. View your hero's profile, including image, powers, backstory, and weakness.

## API Actions

- `start`: Returns project info and available actions.
- `generate`: Generates a superhero profile. Parameters:
  - `name` (optional): Superhero name.
  - `theme` (optional): Theme/archetype (e.g., Tech, Magic, Cosmic).
